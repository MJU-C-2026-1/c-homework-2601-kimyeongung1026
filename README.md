  1. **문제 2:** ... main 함수 내부에 장바구니 담기, 결제, 영수증 출력 등의 모든 기능이 집중되어 코드를 분석하고 디버깅하기 어려워지는 가독성 문제 발생
     * **해결:** ... 핵심 제어 흐름만 main함수에 남겨두고, 독립적으로 수행 가능한 영수증 출력과 할인 금액 연산 로직을 각각 별도의 함수로 독립시킴. 이 과정에서 함수가 요구하는 배열 자료형을 인자로 안전하게 전달하여 모듈화 프로그래밍의 기초를 확립함
